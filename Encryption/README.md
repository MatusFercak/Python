-[pri nesprávnom zadaní argumentov vráti ´chyba´]
<br/>

### Použitie:
```
 python3 encryption <-s ='sifrovanie'/-d ='desifrovanie'> -p <kluc> <subor /.txt> 
```

### Vzorový zápis:
```
python3 encryption.py -s -p key text.txt
```
